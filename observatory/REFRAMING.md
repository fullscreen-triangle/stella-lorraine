### Core Concepts

#### 1. Phase, Not Time
We measure **phase relationships** (angles, in radians) between oscillators, not absolute time intervals.

```
θ_ij = phase difference between oscillators i and j
φ_i(t) = phase of oscillator i at time t

What we measure: Δφ = φ_i - φ_j (dimensionless angle)
NOT: Δt (time interval)
```

#### 2. Harmonic Multiplication
When we extract harmonics, we're measuring phase at higher frequencies:

```
Fundamental: f_0 = 70 THz
100th harmonic: f_100 = 7000 THz

We measure: Phase alignment at 7000 THz
NOT: Time intervals at 1/7000 THz = 0.14 femtoseconds
```

#### 3. Frequency Domain Coordinates
We use frequency ratios as **temporal coordinates**, not time measurements:

```
Temporal Coordinate = (f_1/f_ref, f_2/f_ref, ..., f_n/f_ref)

This is a coordinate system in frequency space
NOT a measurement of absolute time
```

#### 4. Phase-Locking Networks
Our "precision enhancement" is actually **phase synchronization topology**:

```
Network nodes = Oscillatory states
Network edges = Phase-locking relationships
"Precision" = Phase resolution = How finely we can distinguish states
```
