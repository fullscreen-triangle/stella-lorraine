## Figure Specifications

All figures are generated with:
- **DPI:** 300 (publication quality)
- **Format:** PNG
- **Size:** 14" × 10" (2-panel layout) or appropriate for content
- **Style:** Professional scientific plots with:
  - Grid lines (alpha=0.3)

## Data Format Standards

### JSON Files
- **Timestamp:** ISO format (YYYYMMDD_HHMMSS)
- **Units:** Always specified in field names (e.g., `_nK`, `_fs`, `_pK`)
- **Precision:** Scientific notation where appropriate
- **Structure:** Hierarchical, self-documenting

### Figures
- **Panel Labels:** A, B, C, D (top-left of each subplot)
- **Line Styles:**
  - Solid (`-`) for primary data
  - Dashed (`--`) for theoretical/comparison
  - Dotted (`:`) for limits/thresholds
- **Markers:** Different shapes for different datasets
- **Colors:** Consistent meaning across figures
  - Blue: Conventional methods
  - Red: Categorical methods
  - Green: Improvements/advantages
  - Black: Theoretical/reference

## Extending Outputs

To add new outputs to any script:

1. **Add JSON fields:**
   ```python
   results['new_section'] = {
       'parameter': value,
       'metric': measurement
   }
   ```

2. **Add figure panels:**
   ```python
   ax = axes[row, col]
   ax.plot(x, y)
   ax.set_xlabel('...')
   ax.set_ylabel('...')
   ax.set_title('Panel Label) Description')
   ```

3. **Save both:**
   ```python
   # JSON
   with open(output_path, 'w') as f:
       json.dump(results, f, indent=2)

   # Figure
   plt.savefig(fig_path, dpi=300, bbox_inches='tight')
   ```
